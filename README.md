# Prime Credence Solutions | Automação e Auditoria Financeira

Este repositório contém uma solução automatizada para a gestão de esteiras de crédito veicular, desenvolvida para otimizar a análise documental, mitigar riscos operacionais e garantir visibilidade sobre o status de propostas em tempo real.

##  Sobre o Projeto
O Prime Credence Solutions atua como uma camada de inteligência sobre dados financeiros brutos, transformando-os em ativos estratégicos através de:

- **Auditoria Automatizada:** Identificação imediata de gargalos operacionais (propostas paradas vs. em andamento).
- **Gestão de Risco:** Lógica de semáforo de criticidade para processos parados há mais de 15 dias.
- **Padronização de Saídas:**
    - **Relatório Executivo (PDF):** Documentação profissional formatada e pronta para apresentação.
    - **Integração Sistêmica (JSON):** Dados estruturados para integração com ferramentas de BI e sistemas legados.

##  **Ferramentas Utilizadas**
- **Linguagem:** Python
- **Manipulação de Dados:** Pandas
- **Geração de Relatórios:** FPDF
- **Integração:** JSON
